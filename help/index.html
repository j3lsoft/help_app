A) Basic Information

  • Environment Setup - Please visit this link and setup ionic CLI https://ionicframework.com/docs/intro/cli
  • Run our app with Expo - Open ionic-SocialMedia folder on your selected IDE (VSCode is Recommended). Then follow below step.
    • Run npm install command in terminal.

      • Note: If you get error in npm install then please try yarn install or npm install --legacy-peer-deps.

    • For run android app you need to set your android sdk path at ionic-SocialMedia/android/local.properties at line number 8.
    • Run ionic capacitor run android -l --external command in terminal for run app for android.
    • Run ionic capacitor run ios --livereload --external command in terminal for run app for iOS.

      • For more about run ionic app please visit this link:https://ionicframework.com/docs/cli/commands/capacitor-run

  • Change App Icon- https://capacitorjs.com/docs/guides/splash-screens-and-icons
  • Build & Release Android App- https://ionic.io/blog/building-and-releasing-your-capacitor-android-app
  • Build & Release iOS App- https://ionic.io/blog/building-and-releasing-your-capacitor-ios-app

B) Directories & File Uses

  • src - Contain all images, fonts, screen code, services, environment variable, etc. in this folder
    • app(Folder) - Contain all screens code, services, components in this folder.
      • screens(Folder) - Contain all screens code in this folder.
    • assets(Folder) - Contain all images & fonts in this folder.
    • environments(Folder) - Contain all environments variable in this folder. (For production & Debug).
    • theme(Folder) - Contain all files regarding theme in this folder
